# Aadhaar Data Analysis Hackathon

This repository contains Python (Google Colab) notebooks used for analyzing UIDAI Aadhaar datasets.

## Datasets Covered
1. Aadhaar Enrolment Dataset
2. Aadhaar Demographic Update Dataset
3. Aadhaar Biometric Update Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Note
The datasets used are publicly provided by UIDAI in aggregated form.
